# Rust Arena

Rust sandbox

- The book: https://doc.rust-lang.org/book/
- Cargo hub: https://crates.io/
- [Rust principles](./PRINCIPLE.md)

## Install

```bash
curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```

## Create a new program (with Cargo)

```bash
cargo create foo
```
